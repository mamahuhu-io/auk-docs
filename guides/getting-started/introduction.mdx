Auk is a free and open-source, **local-first** API development platform to help you build, test, and document APIs faster. Run it as a desktop app with no mandatory cloud account.

Find articles, guides, and tutorials to help you get started with Auk.

## Try Auk

Download the desktop app from [GitHub Releases](https://github.com/mamahuhu-io/auk/releases/latest) and start sending requests locally in minutes. No login required.

[Learn more](/documentation/getting-started/installation)

## Features

- History
- Collections
- Environments
- Workspaces
- Code Snippets
- Share Requests

- Visit [documentation tab](/documentation/getting-started/introduction) for full list of features.
